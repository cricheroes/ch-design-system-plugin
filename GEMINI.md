# CricHeroes Design System — Gemini context

Building CricHeroes UI in this project? Follow the CricHeroes design system.

- Full guide: `skills/cricheroes-design/SKILL.md`
- Agent rules (shared across tools): `AGENTS.md`
- Tokens to link: `skills/cricheroes-design/colors_and_type.css`

The one rule: **Red = identity. Teal = action. Don't mix them.** Use the tokens,
never hardcode. Build from `skills/cricheroes-design/reference/components.jsx`.
Voice: short, warm, sentence case, no emoji, always "CricHeroes".
