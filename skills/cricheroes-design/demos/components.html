Components

The interactive building blocks every CricHeroes screen relies on: buttons, chips, fields, tabs, the navbar, overlays, badges and loaders. Every colour reaches through a token; red is identity, teal is action.

Buttons Chips Inputs & search Tabs Navbar Overlays Badges Loaders
One rule runs through all of it: red = identity, teal = action.

Teal fills every primary button, link, active tab and focus ring. Red is reserved for the logo, live moments and destructive actions, never a forward-action CTA. One primary CTA per view.

01 · Buttons

Eight variants, three sizes

Colour is chosen by job, not by look: teal leads the primary action, grey is the quieter sibling, red is destructive only. Each variant is shown at Small 32, Medium 48 and Large 56.

02 · Chips

Assist chips and the status hierarchy

Status chips are one cohesive tonal family, ranked by how much attention the state deserves. No solid fills, no glyphs, no pulsing dot: the label text is the differentiator, so Live and Ongoing are intentionally identical.

Status hierarchy
1
Live
Real-time match. Tonal red carries the urgency. --fill-primary-muted · --label-primary-high
2
Ongoing
Tournament running now. Identical to Live; context differentiates. --fill-primary-muted · --label-primary-high
3
Upcoming
Forward-looking, so it moves to the teal family. --fill-secondary-low · --label-secondary-high
4
Cancelled
A deliberate exception, ranked above Completed. Red, but softer. --fill-primary-muted · --label-primary-medium
5
Completed
Expected end state. Calm, archival grey. --fill-grey-low · --label-grey-medium
6
Abandoned
Never concluded (rain, light). Faded so it scans past. --fill-grey-muted · --label-grey-medium
Assist chips

Informational metadata and category labels. Rectangular (--radius-sm), tonal grey, sentence case, with an optional leading glyph. Interactive, so cursor and focus are designed.

Radius is the tell: status chips are full pills, assist and category chips are 8 px rectangles.
