Design system
Foundations

Your cricket matters.

The building blocks every CricHeroes screen shares: colour roles, the type scale, spacing, radius and elevation. Use the tokens, never hardcode.

Red = identity. Teal = action. Don't mix them.

Scarlet red is the logo, live badges and destructive actions, never a normal button fill. Seafoam teal is every action: buttons, links, tabs and focus rings.

Colour roles

Semantic, not decorative. Opacity is preserved through rgb(var(--rgb-*) / a).
