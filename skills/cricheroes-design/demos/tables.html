CricHeroes · design system

Stat tables, built on one primitive

Five cricket tables: a batting scorecard, a bowling scorecard, fall of wickets, the MVP leaderboard and a points table. Every one is the same StatTable. Only the columns change.

Batting Bowling Fall of wickets MVP Points table

The primitive

One StatTable, many configs

This is the shared base. Numeric cells are right-aligned and tabular, the first column is sticky and emphasised, and the whole table sits in an overflow-x wrapper so it scrolls on narrow screens without breaking the page. The variants below add only columns and a few modifiers.

  • Sticky, emphasised first column holds the player, team or label while stats scroll.
  • Right-aligned tabular numerics via .ch-display-numeric so digits line up column to column.
  • Header band in uppercase label-small, plus 1px row dividers from a single token.
  • Designed states: row hover tint, teal focus ring on the scroll region and every link.
  • Modifiers only: .col-emph, .num-strong, .qualified, footer rows. No per-table CSS.
